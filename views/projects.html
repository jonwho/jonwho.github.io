Augmonted

Voted 1st place for best app in Mobile Development class at UCSD. A mobile game app that utilizes Qualcomm's Vuforia for Augmented Reality! Using Unity and written in C# and JavaScript this game makes full use of the tech stack I have learned at UC San Diego. I managed the entire code base for the app using git and merging the work of others and keeping working branches clean. We covered front-end, middle-end, and back-end development. Front-end development is done with Google's Material Design. Middle-end is written in PHP on our server to intercept requests. Back-end development we have a MySQL database on our Amazon EC2 instance. Visit the team site to see our documentation or watch our video for a demo.

Speedy Type

A typing application made for those afflicted by Locked-in Syndrome (LS). This application uses Microsoft Surface and EyeTribe technology to enable typing using only your eyes. The traditional method for communicating with a LS patient is to use an alphabet chart and wait for the patient to blink on letters when a care taker crosses the correct letter. This method does not allow for a patient to initiate conversation. With the use of Speedy Type a patient can initiate conversation and convey needs or simply thoughts. Our method proved to be faster than the traditional method and with my team (UC SpeedyDragons) we made features like piano play and article writing possible. To learn more about Speedy Type and its application read our research paper here and watch our demo video.

Scott the Intern

Playable game on Xbox or PC! Written in C# using XNA framework 3.0 this game makes use of the 2D environment made by stitching together sprite sheets. Gameplay is similar to that of Zelda for free world movement. But the battles are handled like Final Fantasy! Handled using turn-based system with attacks like slaps and coffee bombs! This was a fun project made with a team of 5 and we covered everything from graphics, save states, to audio playbacks. Learned a great deal about software development with a team and an ever growing respect for game developers ever since. I take you 8-bit games for granted no more! Coin Flip

This was the final assignment done for my Web Client class. Working with a team we built a mock coin/buillion exchange market web application. Prices are fetched in real-time using COMEX prices and APIs from Quandl. In the web application you'll find your profile saved to either your email or saved to a third party authorization service like Facebook or Google. After authentication you'll reach the exchange services and you can build a portfolio of coins/buillions ranging from Gold, Silver, and Platinum. You'll find charts tracking prices from 5 days, 1 month, 3 months, and up to a year. This web application was built using JavaScript mainly and some jQuery when helpful.
